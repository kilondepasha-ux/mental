
# Youth Mental Health Campaign (YMHC) — Coursework Website

## Quick Start (VS Code)
1. Extract this zip.
2. Open the folder in **VS Code**.
3. Install the **Live Server** extension (Ritwick Dey).
4. Right‑click `index.html` → **Open with Live Server**.
5. Validate all five pages: Home, About, Resources, Gallery, Contact.

## Deploy (GitHub Pages quick path)
1. Create a GitHub repo and push this folder (contents at the root).
2. Repo **Settings → Pages → Source: `main` / root**.
3. Wait for Pages to build; your site will be live at `https://<username>.github.io/<repo>/`.
